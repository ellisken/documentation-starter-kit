# Installation

## Using pnpm (Recommended)

```bash copy
pnpm install
```

## Using npm

```bash copy
npm install
```

## Using yarn

```bash copy
yarn install
```

## Development Server

Start the local development server with hot reloading:

```bash copy
pnpm dev
```

Your site will be available at `http://localhost:3000`.

## Production Build

To create an optimized production build:

```bash copy
pnpm build
pnpm start
```
