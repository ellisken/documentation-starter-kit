# Introduction

Welcome to the Documentation Starter Kit! This is a ready-to-use docs site built with [Nextra](https://nextra.site) and deployable to [Vercel](https://vercel.com).

## Features

- **MDX Support** — Write content in Markdown with embedded React components
- **Automatic Sidebar** — Navigation generated from your file structure and `_meta.json`
- **Full-text Search** — Built-in search across all your pages
- **Syntax Highlighting** — Code blocks with line numbers, highlighting, and copy button
- **Dark Mode** — Automatic light/dark theme switching
- **Responsive** — Mobile-friendly out of the box

## Quick Start

```bash copy
git clone https://github.com/ellisken/documentation-starter-kit.git
cd documentation-starter-kit
pnpm install
pnpm dev
```

Head to the [Getting Started](/getting-started) section for a full walkthrough.
