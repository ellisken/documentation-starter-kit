# Getting Started

Welcome to the documentation starter kit! This section will help you get up and running quickly.

## Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/) 16.x or later
- [pnpm](https://pnpm.io/), [npm](https://www.npmjs.com/), or [yarn](https://yarnpkg.com/)
- A code editor like [VS Code](https://code.visualstudio.com/)

## Quick Start

Clone the repository and install dependencies:

```bash copy
git clone https://github.com/ellisken/documentation-starter-kit.git
cd documentation-starter-kit
pnpm install
```

Then start the development server:

```bash copy
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see your docs site.
